Detailed Description

Editor helper for hiding and showing a group of GUI elements. Call order in OnInspectorGUI should be:

  • Begin
  • Header/HeaderLabel (optional)
  • BeginFade
  • [your gui elements] (if BeginFade returns true)
  • End
