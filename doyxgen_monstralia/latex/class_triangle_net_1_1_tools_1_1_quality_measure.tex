\doxysubsection{Detailed Description}
Provides mesh quality information. 

Given a triangle abc with points A (ax, ay), B (bx, by), C (cx, cy).

The side lengths are given as a = sqrt((cx -\/ bx)$^\wedge$2 + (cy -\/ by)$^\wedge$2) -- side BC opposite of A b = sqrt((cx -\/ ax)$^\wedge$2 + (cy -\/ ay)$^\wedge$2) -- side CA opposite of B c = sqrt((ax -\/ bx)$^\wedge$2 + (ay -\/ by)$^\wedge$2) -- side AB opposite of C

The angles are given as ang\+\_\+a = acos((b$^\wedge$2 + c$^\wedge$2 -\/ a$^\wedge$2) / (2 $\ast$ b $\ast$ c)) -- angle at A ang\+\_\+b = acos((c$^\wedge$2 + a$^\wedge$2 -\/ b$^\wedge$2) / (2 $\ast$ c $\ast$ a)) -- angle at B ang\+\_\+c = acos((a$^\wedge$2 + b$^\wedge$2 -\/ c$^\wedge$2) / (2 $\ast$ a $\ast$ b)) -- angle at C

The semiperimeter is given as s = (a + b + c) / 2

The area is given as D = abs(ax $\ast$ (by -\/ cy) + bx $\ast$ (cy -\/ ay) + cx $\ast$ (ay -\/ by)) / 2 = sqrt(s $\ast$ (s -\/ a) $\ast$ (s -\/ b) $\ast$ (s -\/ c))

The inradius is given as r = D / s

The circumradius is given as R = a $\ast$ b $\ast$ c / (4 $\ast$ D)

The altitudes are given as alt\+\_\+a = 2 $\ast$ D / a -- altitude above side a alt\+\_\+b = 2 $\ast$ D / b -- altitude above side b alt\+\_\+c = 2 $\ast$ D / c -- altitude above side c

The aspect ratio may be given as the ratio of the longest to the shortest edge or, more commonly as the ratio of the circumradius to twice the inradius ar = R / (2 $\ast$ r) = a $\ast$ b $\ast$ c / (8 $\ast$ (s -\/ a) $\ast$ (s -\/ b) $\ast$ (s -\/ c)) = a $\ast$ b $\ast$ c / ((b + c -\/ a) $\ast$ (c + a -\/ b) $\ast$ (a + b -\/ c))

Determines the bandwidth of the coefficient matrix. 

\begin{DoxyReturn}{Returns}
Bandwidth of the coefficient matrix.
\end{DoxyReturn}


The quantity computed here is the \char`\"{}geometric\char`\"{} bandwidth determined by the finite element mesh alone.

If a single finite element variable is associated with each node of the mesh, and if the nodes and variables are numbered in the same way, then the geometric bandwidth is the same as the bandwidth of a typical finite element matrix.

The bandwidth M is defined in terms of the lower and upper bandwidths\+:

M = ML + 1 + MU

where

ML = maximum distance from any diagonal entry to a nonzero entry in the same row, but earlier column,

MU = maximum distance from any diagonal entry to a nonzero entry in the same row, but later column.

Because the finite element node adjacency relationship is symmetric, we are guaranteed that ML = MU.
