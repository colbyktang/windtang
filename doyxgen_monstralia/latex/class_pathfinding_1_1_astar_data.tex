\doxysubsection{Detailed Description}
Stores the navigation graphs for the A$\ast$ \mbox{\hyperlink{namespace_pathfinding}{Pathfinding}} System. 

An instance of this class is assigned to \mbox{\hyperlink{class_astar_path_a862c787d58adaf4c7b87b37a0c1aea83}{Astar\+Path.\+data}}, from it you can access all graphs loaded through the \mbox{\hyperlink{class_pathfinding_1_1_astar_data_a265fb9535907c9ec56ec6113543e52ff}{graphs}} variable.~\newline
This class also handles a lot of the high level serialization.
