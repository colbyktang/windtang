\doxysubsection{Detailed Description}
Stores temporary node data for a single pathfinding request. Every node has one \mbox{\hyperlink{class_pathfinding_1_1_path_node}{Path\+Node}} per thread used. It stores e.\+g G score, H score and other temporary variables needed for path calculation, but which are not part of the graph structure.
