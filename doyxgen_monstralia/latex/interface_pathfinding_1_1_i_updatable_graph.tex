Updates an area using the specified \mbox{\hyperlink{class_pathfinding_1_1_graph_update_object}{Graph\+Update\+Object}}. 

Notes to implementators. This function should (in order)\+:
\begin{DoxyEnumerate}
\item Call o.\+Will\+Update\+Node on the G\+UO for every node it will update, it is important that this is called B\+E\+F\+O\+RE any changes are made to the nodes.
\item Update walkabilty using special settings such as the use\+Physics flag used with the \mbox{\hyperlink{class_pathfinding_1_1_grid_graph}{Grid\+Graph}}.
\item Call Apply on the G\+UO for every node which should be updated with the G\+UO.
\item Update connectivity info if appropriate (Grid\+Graphs updates connectivity, but most other graphs don\textquotesingle{}t since then the connectivity cannot be recovered later).
