\doxysubsection{Detailed Description}
Linearly interpolating movement script. This movement script will follow the path exactly, it uses linear interpolation to move between the waypoints in the path. This is desirable for some types of games. It also works in 2D. 

See\+: You can see an example of this script in action in the example scene called Example15\+\_\+2D.\hypertarget{class_pathfinding_1_1_a_i_lerp_rec}{}\doxysubsection{Configuration}\label{class_pathfinding_1_1_a_i_lerp_rec}
\hypertarget{class_pathfinding_1_1_a_i_lerp_rec-snapped}{}\doxysubsubsection{Recommended setup for movement along connections}\label{class_pathfinding_1_1_a_i_lerp_rec-snapped}
This depends on what type of movement you are aiming for. If you are aiming for movement where the unit follows the path exactly and move only along the graph connections on a grid/point graph. I recommend that you adjust the \mbox{\hyperlink{class_pathfinding_1_1_start_end_modifier}{Start\+End\+Modifier}} on the \mbox{\hyperlink{class_pathfinding_1_1_seeker}{Seeker}} component\+: set the \textquotesingle{}Start Point Snapping\textquotesingle{} field to \textquotesingle{}Node\+Connection\textquotesingle{} and the \textquotesingle{}End Point Snapping\textquotesingle{} field to \textquotesingle{}Snap\+To\+Node\textquotesingle{}. \mbox{[}Open online documentation to see images\mbox{]} \mbox{[}Open online documentation to see images\mbox{]}\hypertarget{class_pathfinding_1_1_a_i_lerp_rec-smooth}{}\doxysubsubsection{Recommended setup for smooth movement}\label{class_pathfinding_1_1_a_i_lerp_rec-smooth}
If you on the other hand want smoother movement I recommend setting \textquotesingle{}Start Point Snapping\textquotesingle{} and \textquotesingle{}End Point Snapping\textquotesingle{} to \textquotesingle{}Closest\+On\+Node\textquotesingle{} and to add the Simple Smooth Modifier to the Game\+Object as well. Alternatively you can use the Pathfinding.\+Funnel\+Modifier Funnel which works better on navmesh/recast graphs or the \mbox{\hyperlink{class_pathfinding_1_1_raycast_modifier}{Pathfinding.\+Raycast\+Modifier}}.

You should not combine the Simple Smooth Modifier or the \mbox{\hyperlink{class_pathfinding_1_1_funnel}{Funnel}} Modifier with the Node\+Connection snapping mode. This may lead to very odd behavior.

\mbox{[}Open online documentation to see images\mbox{]} \mbox{[}Open online documentation to see images\mbox{]} You may also want to tweak the \mbox{\hyperlink{class_pathfinding_1_1_a_i_lerp_a4d52abbb0392e6c70067a8c311af7d3a}{rotation\+Speed}}.
