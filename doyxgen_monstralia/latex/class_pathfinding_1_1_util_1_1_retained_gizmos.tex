\doxysubsection{Detailed Description}
Helper for drawing Gizmos in a performant way. This is a replacement for the Unity Gizmos class as that is not very performant when drawing very large amounts of geometry (for example a large grid graph). These gizmos can be persistent, so if the data does not change, the gizmos do not need to be updated. 

How to use
\begin{DoxyItemize}
\item Create a \mbox{\hyperlink{struct_pathfinding_1_1_util_1_1_retained_gizmos_1_1_hasher}{Hasher}} object and hash whatever data you will be using to draw the gizmos Could be for example the positions of the vertices or something. Just as long as if the gizmos should change, then the hash changes as well.
\item Check if a cached mesh exists for that hash
\item If not, then create a \mbox{\hyperlink{class_pathfinding_1_1_util_1_1_retained_gizmos_1_1_builder}{Builder}} object and call the drawing methods until you are done and then call Finalize with a reference to a gizmos class and the hash you calculated before.
\item Call gizmos.\+Draw with the hash.
\item When you are done with drawing gizmos for this frame, call gizmos.\+Finalize\+Draw
\end{DoxyItemize}
