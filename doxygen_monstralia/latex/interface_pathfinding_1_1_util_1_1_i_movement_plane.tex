\doxysubsection{Detailed Description}
Transforms to and from world space to a 2D movement plane. The transformation is guaranteed to be purely a rotation so no scale or offset is used. This interface is primarily used to make it easier to write movement scripts which can handle movement both in the XZ plane and in the XY plane.
