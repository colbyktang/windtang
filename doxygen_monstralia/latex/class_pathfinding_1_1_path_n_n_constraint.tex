A \mbox{\hyperlink{class_pathfinding_1_1_path_n_n_constraint}{Path\+N\+N\+Constraint}} can be assigned to the \mbox{\hyperlink{class_pathfinding_1_1_path_a0fecce33ad5424d6e0bf0ec4b58bbac5}{Path.\+nn\+Constraint}} field, the path will first search for the start node, then it will call \mbox{\hyperlink{class_pathfinding_1_1_path_n_n_constraint_a5eea657d76f45567e849c2405055786a}{Set\+Start}} and proceed with searching for the end node (nodes in the case of a Multi\+Target\+Path).~\newline
The default \mbox{\hyperlink{class_pathfinding_1_1_path_n_n_constraint}{Path\+N\+N\+Constraint}} will constrain the end point to lie inside the same area as the start point.
