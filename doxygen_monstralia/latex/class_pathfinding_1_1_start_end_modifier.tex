\doxysubsection{Detailed Description}
Adjusts start and end points of a path. 

This modifier is included in the \mbox{\hyperlink{class_pathfinding_1_1_seeker}{Pathfinding.\+Seeker}} component and is always used if you are using a \mbox{\hyperlink{class_pathfinding_1_1_seeker}{Seeker}}. When a path is calculated the resulting path will only be the positions of the nodes it passes through. However often you may not want to navigate to the center of a specific node but instead to a point on the surface of a node. This modifier will adjust the endpoints of the path.
