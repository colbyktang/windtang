\doxysubsection{Detailed Description}
Internal Representation of a Tween~\newline
 ~\newline
 This class represents all of the optional parameters you can pass to a method (it also represents the internal representation of the tween).~\newline
~\newline
 {\bfseries{Optional Parameters}} are passed at the end of every method\+:~\newline
 ~\newline
 ~~{\itshape Example\+:}~\newline
 ~~Lean\+Tween.\+move\+X( game\+Object, 1f, 1f).set\+Ease( \href{LeanTweenType.html}{\texttt{ Lean\+Tween\+Type}}.ease\+In\+Quad ).set\+Delay(1f);~\newline
 ~\newline
 You can pass the optional parameters in any order, and chain on as many as you wish.~\newline
 You can also {\bfseries{pass parameters at a later time}} by saving a reference to what is returned.~\newline
 ~\newline
 Retrieve a {\bfseries{unique id}} for the tween by using the \char`\"{}id\char`\"{} property. You can pass this to Lean\+Tween.\+pause, Lean\+Tween.\+resume, Lean\+Tween.\+cancel, \mbox{\hyperlink{class_lean_tween_a325405079311587dfad028a14d510f58}{Lean\+Tween.\+is\+Tweening}} methods~\newline
 ~\newline
 ~~\doxysubparagraph*{Example\+:}

~~int id = Lean\+Tween.\+move\+X(game\+Object, 1f, 3f).id;~\newline
 ~~// pause a specific tween ~~Lean\+Tween.\+pause(id);~\newline
 ~~// resume later ~~Lean\+Tween.\+resume(id);~\newline
 ~~// check if it is tweening before kicking of a new tween ~~if( Lean\+Tween.\+is\+Tweening( id ) )\{~\newline
 ~~ ~~ Lean\+Tween.\+cancel( id );~\newline
 ~~ ~~ Lean\+Tween.\+move\+Z(game\+Object, 10f, 3f);~\newline
 ~~\}~\newline
