\doxysubsection{Detailed Description}
Defines a shape for a \mbox{\hyperlink{class_pathfinding_1_1_graph_update_object}{Pathfinding.\+Graph\+Update\+Object}}. The shape consists of a number of points which it can either calculate the convex hull of or use as a polygon directly. 

A shape is essentially a 2D shape however it can be rotated arbitrarily. When a matrix and a list of points is specified in the constructor the matrix decides what direction is the \textquotesingle{}up\textquotesingle{} direction. When checking if a point is contained in the shape, the point will be projected down on a plane where the \textquotesingle{}up\textquotesingle{} direction is the normal and then it will check if the shape contains the point.
