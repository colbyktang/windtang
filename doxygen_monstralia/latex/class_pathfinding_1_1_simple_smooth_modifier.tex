\doxysubsection{Detailed Description}
Modifier which smooths the path. This modifier can smooth a path by either moving the points closer together (Simple) or using Bezier curves (Bezier).~\newline


Attach this component to the same Game\+Object as a \mbox{\hyperlink{class_pathfinding_1_1_seeker}{Seeker}} component. ~\newline
This component will hook in to the \mbox{\hyperlink{class_pathfinding_1_1_seeker}{Seeker}}\textquotesingle{}s path post-\/processing system and will post process any paths it searches for. Take a look at the Modifier Priorities settings on the \mbox{\hyperlink{class_pathfinding_1_1_seeker}{Seeker}} component to determine where in the process this modifier should process the path. ~\newline
~\newline
Several smoothing types are available, here follows a list of them and a short description of what they do, and how they work. But the best way is really to experiment with them yourself.~\newline

\begin{DoxyItemize}
\item {\bfseries{Simple}} Smooths the path by drawing all points close to each other. This results in paths that might cut corners if you are not careful. It will also subdivide the path to create more more points to smooth as otherwise it would still be quite rough. \mbox{[}Open online documentation to see images\mbox{]}
\item {\bfseries{Bezier}} Smooths the path using Bezier curves. This results a smooth path which will always pass through all points in the path, but make sure it doesn\textquotesingle{}t turn too quickly. \mbox{[}Open online documentation to see images\mbox{]}
\item {\bfseries{Offset\+Simple}} An alternative to Simple smooth which will offset the path outwards in each step to minimize the corner-\/cutting. But be careful, if too high values are used, it will turn into loops and look really ugly.
\item {\bfseries{Curved Non Uniform}} \mbox{[}Open online documentation to see images\mbox{]}
\end{DoxyItemize}

Note\+: Modifies vector\+Path array T\+O\+DO\+: Make the smooth modifier take the world geometry into account when smoothing
