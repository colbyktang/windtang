{\footnotesize\ttfamily void Pathfinding.\+I\+Work\+Item\+Context.\+Queue\+Flood\+Fill (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Call during work items to queue a flood fill. An instant flood fill can be done via Flood\+Fill() but this method can be used to batch several updates into one to increase performance. Work\+Items which require a valid Flood Fill in their execution can call Ensure\+Valid\+Flood\+Fill to ensure that a flood fill is done if any earlier work items queued one. 

Once a flood fill is queued it will be done after all Work\+Items have been executed.

Deprecated\+: Avoid using. This will force a full recalculation of the connected components. In most cases the \mbox{\hyperlink{class_pathfinding_1_1_hierarchical_graph}{Hierarchical\+Graph}} class takes care of things automatically behind the scenes now.
