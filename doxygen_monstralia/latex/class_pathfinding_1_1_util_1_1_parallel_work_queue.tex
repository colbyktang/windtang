\doxysubsection{Detailed Description}
Helper for parallelizing tasks. More specifically this class is useful if the tasks need some large and slow to initialize \textquotesingle{}scratch pad\textquotesingle{}. Using this class you can initialize a scratch pad per thread and then use the appropriate one in the task callback (which includes a thread index). 

Any exception that is thrown in the worker threads will be propagated out to the caller of the \mbox{\hyperlink{class_pathfinding_1_1_util_1_1_parallel_work_queue_a770da4cc8f498fdc37c761ca0679c70c}{Run}} method.
