\doxysubsection{Detailed Description}
Abstract class that all Foods inherit from; inherits from \mbox{\hyperlink{class_colorable}{Colorable}}. 

The abstract \mbox{\hyperlink{class_food}{Food}} class handles spawning the food into the scene. Everything else is handled by the individual \mbox{\hyperlink{class_food}{Food}} scripts.
