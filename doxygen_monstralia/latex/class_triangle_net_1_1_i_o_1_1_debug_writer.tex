\doxysubsection{Detailed Description}
Writes a the current mesh into a text file. 

File format\+:

num\+\_\+nodes id\+\_\+1 nx ny mark ... id\+\_\+n nx ny mark

num\+\_\+segs id\+\_\+1 p1 p2 mark ... id\+\_\+n p1 p2 mark

num\+\_\+tris id\+\_\+1 p1 p2 p3 n1 n2 n3 ... id\+\_\+n p1 p2 p3 n1 n2 n3
