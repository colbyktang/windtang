Detailed Description

GraphModifier is used for modifying graphs or processing graph data based on events. This class is a simple container for a number of events.

Warning: Some events will be called both in play mode and in editor mode (at least the scan events). So make sure your code handles both cases well. You may choose to ignore editor events. See: Application.IsPlaying
